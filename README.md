# 星落虚空回响 (StarFall VoidEchoe)

一款基于 **Three.js** 与 **原生 JavaScript** 开发的 HTML5 3D 游戏。

## 项目概览
本项目是基于 MIT 开源协议创作的 **StarFall** 分支版本，主要玩法与内容更新与原版保持一致，并在此基础上进行了自定义优化与改动。
> **提示：** 本项目可视为 StarFall 的“C服”（区别于官方 M 服）。
> 
 * **作者**: MiaoSL
 * **分支维护**: clcwpwqi
 * **技术栈**: Three.js, JavaScript
 * **开源协议**: MIT 许可证

## 600Final 版本日志
### 核心优化
 * **性能提升**: 进行大量内容重构，实现极致性能优化，新增两档兼容模式，广泛适配各类机型。
 * **地图拓展**: 地图规模扩增至近乎无限。
 * **交互优化**: 优化对话系统，支持点击切换，并调整了“自动模式”为打字效果后延迟 1s 自动跳转。
### 画面与环境
 * **景观增强**: 添加树木显示及多颜色树木；优化地面颜色渐变。
 * **视觉氛围**: 添加远处雾效、天空颜色渐变，画面表现力大幅提升。
 * **细节处理**: 去除草地的阴影，修复地形显示异常。
### 体验修复
 * **操作手感**: 加入平滑镜头技术，优化时间流转与角色运动表现。
 * **BUG 修复**: 修复了角色平滑镜头颤动问题，修复了少量已知“史山”代码，并（预留性地）添加了大量未知“史山”。
